## What I'm Building

**[Alochi](https://alochi.org)** — a full-stack education platform serving schools across Uzbekistan.

| Component | Stack | Description |
|---|---|---|
| 🏫 **Backend API** | Django 5 · DRF · Celery | Lessons, attendance, homework, grading |
